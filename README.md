# Hotel-Reservation-System
A smart hotel reservation web app with chatbot interaction, room booking, date-based billing, payment confirmation UI, and ML-based cancellation risk prediction.
## Features
- AI-style hotel chatbot
- Room type selection
- Member name and age input
- Check-in/check-out calendar
- Auto bill calculation
- Payment confirmation
- Machine Learning cancellation prediction
- Flask backend with HTML, CSS, JavaScript frontend

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript
